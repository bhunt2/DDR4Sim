1. Copy project files to desired working directory
	a. The folder structure should be
		i. sim: contains all the necessary files for simulation
		ii. src: contains the design files (system verilog and so forth)

2. Setup project environment in QuestaSim
	a. Open QuestaSim
	b. Create a new empty project with the work directory in the sim folder

3. Compile/Run Simulation
	a. With the work directory in the sim folder you can just type 'do run.do'
	   in the console to run.
	b. Type run to run simulation to completion.
